# Rookie Sideloader Setup Guide

## 🛡 REAL TIME PROTECTION
(Windows Defender Exclusion)

1. Open your **C: Drive**
2. Create a folder named: `RSL`
3. Inside `RSL`, create another folder named: `Rookie`
4. On your PC, search for: **Virus & Threat Protection**
5. Under **Virus & Threat Protection Settings**, click **Manage Settings**
6. Scroll down to **Exclusions**
7. Click **Add or Remove Exclusions**
8. Click **Yes** if prompted
9. Click **+ Add an Exclusion** → select **Folder**
10. Navigate to `C:\`
11. Select the `RSL` folder you created
12. Click **Select Folder**

## 📦 INSTALLING ROOKIE

1. Open the downloaded `.zip` file
2. Copy **ALL files** into:

```C:\RSL\Rookie```

3. Run:

```androidsideloader.exe```


Rookie should now connect to the updated servers using the modified
`vrp-public.json`.

---

## 🖥 OPTIONAL (Shortcut)

1. Right-click `androidsideloader.exe`
2. Click **Create Shortcut**
3. Move the shortcut to your desktop for easy access
